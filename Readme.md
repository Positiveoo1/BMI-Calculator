
# BMI Calculator

A simple application to calculate Body Mass Index (BMI) based on height and weight.

## Features

- Calculate BMI from user input
- Display BMI category (underweight, normal, overweight, obese)
- Simple and intuitive interface

## Installation

Clone the repository and install dependencies:

```bash
git clone <repository-url>
cd BMI-Calculator
```

## Usage

Run the application:

```bash
npm start
```

Enter your height (in cm) and weight (in kg) to calculate your BMI.

## BMI Categories

- Underweight: BMI < 18.5
- Normal weight: 18.5 ≤ BMI < 25
- Overweight: 25 ≤ BMI < 30
- Obese: BMI ≥ 30
